# wechat

seajs模块化

常见的 JavaScript 模块化规范

- CommonJS
  + Node.js
- AMD
  + RequireJS
- CMD Common Module Definition
  + CMD 就是 SeaJS 这个模块加载器在推广的过程中定义的一个模块规范
- ECMAScript
  + ECMAScript 6
- UMD

1. 在页面中引入 sea.js
2. 使用 `define` 函数定义模块
3. 使用 `require` 函数加载模块
4. 使用 `module.exports` 对外暴露接口对象
5. 使用 `seajs.use` 函数启动模块系统
