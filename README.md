# wechat

## 常见的 JavaScript 模块化规范

- CommonJS
  + Node.js
- AMD
  + RequireJS AMD 是提前执行  推崇依赖前置
- CMD Common Module Definition
  + CMD 就是 SeaJS  CMD 是延迟执行 推崇依赖就近
- ECMAScript
  + ECMAScript 6
- UMD

## seajs用法
1. 在页面中引入 sea.js
2. 使用 `define` 函数定义模块
3. 使用 `require` 函数加载模块
4. 使用 `module.exports` 对外暴露接口对象
5. 使用 `seajs.use` 函数启动模块系统
